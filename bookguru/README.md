# BookGuru - Homepage (React + Tailwind)

## What this includes
- React + Vite project
- Tailwind CSS integration
- Framer Motion for lightweight animations
- Forest-green theme and responsive mobile-first layout
- Homepage components: Navbar, Hero, WhatWeOffer, WhyChooseUs, LatestNews, Subscription, Footer
- Images loaded from public URLs (Unsplash)

## Install & Run
1. Extract the project folder.
2. In the project root run:
```bash
npm install
npm run dev
```
3. Open the printed dev server URL (usually http://localhost:5173)

## Notes
- Replace the placeholder images and news links with your real content.
- To build for production: `npm run build`
